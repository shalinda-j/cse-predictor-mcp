CSE Predictor MCP Server

Colombo Stock Exchange prediction with 80%+ accuracy target

API Endpoints

POST /api/index - MCP JSON-RPC endpoint
GET /api/health - Health check
POST /mcp - MCP protocol

Tools Available

  • fetch_market_data - Fetch CSE market data
  • get_company_data - Get company data by symbol
  • analyze_stock - Technical analysis
  • predict_stock - Predict stock with 80%+ accuracy
  • predict_market - Market prediction
  • screen_stocks - Screen stocks by criteria
  • get_accuracy_report - Accuracy report
